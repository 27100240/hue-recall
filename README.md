# Hue Recall

A soft, iOS software inspired color matching game where players study a random color, recreate it using HSL sliders, and get scored with mildly judgmental commentary.

## Overview

Hue Recall is a lightweight interactive web game built to test how accurately users can recreate colors by eye.

In each round, the player is shown a random target color and must rebuild it using hue, saturation, and lightness sliders. After submitting a guess, the game compares the player’s color to the original, calculates a score, and responds with a playful remark. After 3 rounds, the player receives a final average score.

The project was designed with a clean, glass-like interface and a more playful game than a typical UI prototype.

## Features

- 3-round color matching game
- Randomized target colors each round
- Live color preview while adjusting sliders
- HSL-based scoring system
- Per-round score out of 100
- Fun remarks based on score
- Final average score screen
- Soft, glass-like, iOS-style visual design

## Tech Stack

- React
- Vite
- JavaScript
- Custom CSS

## How It Works

1. Start the game
2. View a random target color
3. Recreate it using hue, saturation, and lightness sliders
4. Submit your guess
5. Receive a score and remark
6. Repeat for 3 total rounds
7. View your final average score

## Why I Built This

I wanted to build a small but polished interactive project that felt playful, visual, and satisfying to use. Hue Recall let me work on frontend state management, game logic, score calculation, and interface design while creating something that people could actually enjoy sharing and replaying.
